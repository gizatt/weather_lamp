Weather Lamp
------------

Super WIP.

## Layout

Built around an Arduino MKR Wifi 1010.

Current pin assignments:
* 0: Unused
* 1: Display DC (data mode select) pin
* 2: asdf
* 3: SPI CS, SD Card
* 4: Lightning Detector interrupt
* 5: Unused
* 6: SPI CS, Barometer
* 7: SPI CS, Lightning Detector
* 8: SPI MOSI
* 9: SPI SCK
* 10: SPI MISO
* 11-14: Unusued
